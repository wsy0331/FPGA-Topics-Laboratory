# Ex3 Pingpong
這是一個雙人對戰的乒乓球遊戲，使用 8 顆 LED (`o_led`) 作為球，並透過兩個開關 (`i_swL`, `i_swR`) 進行擊球，在原題目乒乓球對打中加入剛開始可以任意一方發球，且具有完整計分的功能。
